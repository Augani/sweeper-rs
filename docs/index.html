Why Sweeper?

Built from the ground up with performance and safety in mind.

Blazing Fast

Parallel scanning with rayon. Finds gigabytes of reclaimable space in seconds, not minutes.

Memory Safe

Built with Rust. No crashes, no memory leaks, no undefined behavior. Rock solid.

Safe by Default

Moves files to trash instead of permanent deletion. Preview everything before cleanup.

Beautiful UI

GPU-accelerated interface built with GPUI and adabraka-ui. 60fps smooth animations.

Cross Platform

Native support for macOS and Windows. Linux coming soon.

Smart Detection

Confidence scores for each item. Stale project detection. Never accidentally delete active work.
